# 🌊 MarFlow

Tu gestor personal de tareas.

## Desplegar en Vercel (sin código)

### Opción A — Subir carpeta directamente (más fácil)

1. Ve a [vercel.com](https://vercel.com) e inicia sesión (puedes usar Google)
2. Haz clic en **"Add New Project"**
3. Selecciona **"Browse"** o arrastra esta carpeta `marflow`
4. Vercel detectará Vite automáticamente
5. Haz clic en **"Deploy"** → en 1 minuto tendrás tu URL

### Opción B — Via GitHub (recomendado para actualizaciones)

1. Crea una cuenta en [github.com](https://github.com) si no tienes
2. Crea un repositorio nuevo (puede ser privado)
3. Sube esta carpeta al repositorio
4. En [vercel.com](https://vercel.com) → "Add New Project" → conecta tu GitHub
5. Selecciona el repositorio → **Deploy**

Cada vez que actualices el código en GitHub, Vercel se actualiza solo.

## Ejecutar en local

```bash
npm install
npm run dev
```

Abre http://localhost:5173
